Lalamove & GoSend

Kredensial API diisi di file .env lalu restart server. Setelah itu klik Uji API di bawah.

Estimasi cadangan (jika API belum aktif)

Cara mendapatkan kredensial

  1. Lalamove: Daftar di developers.lalamove.com → Partner Portal → Developers → salin API Key & Secret (sandbox: pk_test).
  2. GoSend: Isi formulir di gojek.com/gosend/api → tim GoSend kirim Client ID & Pass Key untuk staging.
  3. Buka file .env di folder proyek, isi variabel LALAMOVE_* dan GOSEND_*.
  4. Restart server (npm run dev) lalu klik tombol Uji API di atas.
